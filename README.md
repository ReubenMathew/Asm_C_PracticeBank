# Asm_C_PracticeBank
Redoing all of my computer science competition solutions from Python,Java,C++ to C &amp; Assembly

To compile and make assembly executable:

1. `nasm -f elf <filename>.asm`

2. `ld -m elf_i386 -s -o <filename> <filename>.o`

3. `./<filename>`
